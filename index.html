상황별 응급약물 참고

근무 중 빠르게 확인하는 임상 Cheat Sheet입니다. 약물 선택 → 사용 이유 → 간호 포인트 → 모니터링 순으로 보세요.

Cardiac Arrest / CPR

사용 약물

MEPI Epinephrine
MCDR Amiodarone

왜 사용하는가

  • 관상동맥 관류압 증가
  • 심장 자극
  • Refractory VF/VT 안정화

간호 핵심

  • CPR 중 투여 시점 확인
  • Flush
  • ECG / BP monitoring

모니터링

  • ECG
  • BP
  • ROSC
  • 상황 인지·우선 처치·재평가: 병원 프로토콜 확인 후 입력 예정
기억 CPR 중 투여 시점 + Flush가 핵심이다.

Stable SVT

사용 약물

MADEN Adenosine

왜 사용하는가

  • AV node 순간 차단
  • 리듬 전환

간호 핵심

  • 빠른 IV push
  • Flush

모니터링

  • ECG
  • HR
  • 상황 인지·우선 처치·재평가: 병원 프로토콜 확인 후 입력 예정
기억 Adenosine은 빠른 IV push가 핵심이다.

Hyperkalemia + ECG 변화

사용 약물

MCAGLU Calcium Gluconate
MBIVON Sodium Bicarbonate

왜 사용하는가

  • 심근세포막 안정화
  • 산증 완화
  • K 세포 내 이동 보조

간호 핵심

  • 칼륨 제거 약물 아님 인지
  • 산증 동반 여부 확인

모니터링

  • ECG
  • K 재검
  • 상황 인지·우선 처치·재평가: 병원 프로토콜 확인 후 입력 예정
기억 Calcium은 칼륨 제거 약물이 아니다.

Septic Shock / Severe Hypotension

사용 약물

MLEVO Norepinephrine
M8DOPAM Dopamine Premix M8 (400mg)
M16DOPAM Dopamine Premix M16 (800mg)

왜 사용하는가

  • 강한 혈관수축 → MAP 유지
  • 혈압 상승
  • perfusion 개선

간호 핵심

  • 농도 / 용량 구분

모니터링

  • MAP
  • BP
  • 상황 인지·우선 처치·재평가: 병원 프로토콜 확인 후 입력 예정
기억 Norepinephrine은 Septic shock의 MAP 유지 승압제이다.

Bradycardia + Hypotension

사용 약물

MDOPA Dopamine

왜 사용하는가

  • HR 상승
  • BP 상승

간호 핵심

  • Tachycardia 관찰

모니터링

  • HR
  • BP
  • 상황 인지·우선 처치·재평가: 병원 프로토콜 확인 후 입력 예정
기억 Dopamine 투여 중 Tachycardia를 관찰한다.

E-cart 약물 구성

부서별 응급카트 비치 현황을 확인하고, 기초 약물 학습 상세로 바로 이동할 수 있습니다.

부서별 응급카트 구성은 환자 특성과 응급상황 발생 빈도를 고려하여 다를 수 있습니다.

※ 대전보훈병원 심폐소생술 관리규정 별첨4(응급카트 약품목록) 기준입니다. 실제 투여 시 최신 프로토콜·처방을 확인하세요.
